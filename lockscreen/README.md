# lockscreen

Фуллскрин-локер на tkinter. Закрывается только паролем.

## Запуск

```bash
python lock.py
```

## Пароль

`1` — захардкожен в начале `lock.py`, константа `PASSWORD`.

## Что делает

- Окно на весь экран, поверх всего, без рамки
- Заблокированы Esc, Alt+F4, Ctrl+W, Ctrl+Q, F11, крестик
- Часы в углу, счётчик попыток, тряска поля при ошибке
- Принудительный фокус на поле ввода каждые 500 мс

## Грабли

- Win+D, Ctrl+Alt+Del, Ctrl+Alt+T (на Linux) системой не блокируются — это уровень ОС, из юзерспейса не закрыть
- Если нужно "по-настоящему" — это уже kiosk mode на уровне DE/групповых политик
- На macOS `overrideredirect` ведёт себя странно, фуллскрин лучше работает через `-fullscreen`
- Убить процесс из tty/диспетчера задач можно — это не зловред, это окно
