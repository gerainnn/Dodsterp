# SMS Service Testing Bot

> **Важно:** Данный проект создан исключительно в образовательных целях для изучения разработки Telegram ботов на Python с использованием библиотеки aiogram. Проект предназначен для тестирования SMS-шлюзов и сервисов авторизации. Использование данного ПО для нарушения законодательства запрещено.

## Описание проекта

Telegram бот для тестирования SMS-сервисов и изучения работы SMS-шлюзов. Проект демонстрирует:
- Архитектуру асинхронных Telegram ботов на aiogram 3.x
- Работу с FSM (Finite State Machine) для управления диалогами
- Интеграцию с внешними API сервисами
- Обработку пользовательского ввода и валидацию данных

## Юридическая информация

⚠️ **ПРЕДУПРЕЖДЕНИЕ:**
- Данный проект создан в образовательных целях
- Массовая отправка SMS без согласия получателя является нарушением законодательства РФ (ст. 138 УК РФ)

## Установка и запуск

### Требования
- Python 3.10+
- pip

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/gerainnn/Dodsterp.git
cd Dodsterp
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

5. Добавьте токен бота в файл `.env`:
```
BOT_TOKEN=your_telegram_bot_token_here
```

6. Запустите бота:
```bash
python bot.py
```

## Архитектура

### FSM States (состояния)
```
SpamStates
├── waiting_for_phone   # Ожидание ввода номера
├── waiting_for_speed   # Выбор скорости
└── spamming            # Процесс отправки
```

### Поток данных
```
User Input → Validation → Speed Selection → SMS Service → Progress Update
```
