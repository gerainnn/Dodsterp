# SMS Bot - Telegram бот для отправки SMS

Telegram бот для отправки SMS-сообщений на номера РФ (+7) через различные SMS-шлюзы.

## Описание проекта

Проект представляет собой Telegram бота, который позволяет отправлять SMS-сообщения на мобильные номера России. Основные возможности:

- 📱 Отправка SMS на номера РФ (+7)
- ⚡ Настраиваемая скорость отправки (1/3/5/10 SMS в секунду)
- 🔄 Интеграция с несколькими SMS-шлюзами
- 📊 Мониторинг статуса отправки в реальном времени

## Установка и запуск

### Требования
- Python 3.10+
- pip

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/gerainnn/Dodsterp.git
cd Dodsterp
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

5. Добавьте токен бота в файл `.env`:
```
BOT_TOKEN=your_telegram_bot_token_here
```

6. Запустите бота:
```bash
python bot.py
```

## Архитектура

### FSM States (состояния)
```
SendStates
├── waiting_for_phone   # Ожидание ввода номера
├── waiting_for_speed   # Выбор скорости
└── sending             # Процесс отправки
```

### Поток данных
```
User Input → Validation → Speed Selection → SMS Service → Progress Update
```

## Известные проблемы

1. В `sms_services.py` используются заглушки вместо реальных API
2. Нет персистентного хранения данных
3. Нет системы авторизации пользователей
4. Отсутствует graceful shutdown
